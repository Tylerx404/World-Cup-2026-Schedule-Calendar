## Overview

This project helps users follow the World Cup 2026 schedule on the web and add the full fixture list to Google Calendar, Apple Calendar, Outlook, or any app that supports iCalendar.

## Main Features

- View the World Cup 2026 schedule on the website.
- Supports two languages: Vietnamese and English.
- Full schedule page organized by tournament rounds.
- Dedicated teams page.
- Add-to-calendar page with subscription options.
- WebCal support for one-time subscription with automatic updates.
- `.ics` file download for manual import.
- Dark mode support.

## Run Locally

```bash
bun install
bun dev
```

Open `http://localhost:3000`.

Other commands:

```bash
bun run lint
bun run build
bun run start
```
