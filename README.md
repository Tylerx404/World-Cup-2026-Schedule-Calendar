## Giới thiệu

Dự án này giúp người dùng theo dõi lịch thi đấu World Cup 2026 trên web và thêm toàn bộ lịch vào Google Calendar, Apple Calendar, Outlook hoặc các ứng dụng hỗ trợ iCalendar.

## Chức năng chính

- Xem lịch thi đấu World Cup 2026 trên website.
- Hỗ trợ hai ngôn ngữ: tiếng Việt và tiếng Anh.
- Trang lịch thi đấu đầy đủ theo từng vòng.
- Trang danh sách đội tuyển tham dự.
- Trang hướng dẫn thêm lịch vào calendar.
- Hỗ trợ WebCal để subscribe một lần và tự động nhận cập nhật.
- Hỗ trợ tải file `.ics` để import thủ công.
- Hỗ trợ dark mode.

## Chạy local

```bash
bun install
bun dev
```

Mở `http://localhost:3000`.

Các lệnh khác:

```bash
bun run lint
bun run build
bun run start
```
